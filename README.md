## Run it locally

Clone the project

```bash

git clone project-name

```

Go to the project directory

```bash

cd project-name

```

### Or you can just download it as zip, then

Install dependencies

`npm install` or `yarn install`

Start the server

`npm run dev` or `yarn dev`

## Dependencies

1. React
2. React-Router
3. Tailwind
4. Vite(Frontend Tooling)
